# Next-intl (다국어 라이브러리)

Next-intl은 웹 사이트의 다국어 처리를 하기 위한 라이브러리이다.

{% hint style="warning" %}
현재 기록날짜  2023-11-20

현재 안정적인 버전이라고하나, 차후 변경될 가능성도 있다.

날짜와 버전에 유의해서 사용하자.

Package version

```json
"next-intl": "^3.1.0",
```
{% endhint %}
